== README

This web app is meant to track the movement of Connective Quotes cards by placing markers on a map in connection with each card's serial number.


About this app:

* Ruby version 2.2.1

* Rails version 4.1.8

* Configuration: deployed with Heroku



If you would like to make enhancements or to streamline the code, please fork this project.

*The Connective Foundation is a non-profit 501c3 organization and all our projects are completely free and open source.*
